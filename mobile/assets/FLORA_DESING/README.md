## Direção visual
A linguagem segue um caminho **editorial, premium, limpo e sereno**:
- fundo off-white suave
- tipografia com presença editorial
- verde profundo como tom principal da marca
- dourado discreto como acento sofisticado
- flor de íris branca como imagem de design integrada, não como elemento decorativo infantil

## Recomendações de uso
- **App mobile:** prefira os PNGs de fundo.
- **Apresentação / handoff / documentação:** use os SVGs.
- **Logo sobre fundo claro:** use `iris-wordmark-dark.svg`.
- **Logo sobre fundo escuro:** use `iris-wordmark-light.svg`.
- **Assinatura institucional:** use os lockups.
